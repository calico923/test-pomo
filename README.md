# 🍅 Pomodoro Timer

シンプルなポモドーロタイマーアプリ（Python + Tkinter）

## 機能

- **25分作業 / 5分休憩**のポモドーロテクニック
- 4セッション完了後は**15分の長い休憩**
- タイマーのスタート/一時停止/リセット/スキップ
- 作業時間・休憩時間のカスタマイズ
- 完了セッション数のカウント
- 視覚的なステータス表示（赤=作業、緑=休憩）

## 必要環境

- Python 3.x
- Tkinter（Python標準ライブラリ）

## 起動方法

```bash
cd pomodoro-app
python pomodoro.py
```

または

```bash
python3 pomodoro.py
```

## 使い方

1. **スタート**: タイマーを開始
2. **一時停止**: タイマーを一時停止
3. **リセット**: 現在のセッションをリセット
4. **スキップ**: 次のセッション（作業/休憩）へスキップ

## 設定

画面下部のスピンボックスで各時間をカスタマイズできます：
- 作業時間（デフォルト: 25分）
- 短い休憩（デフォルト: 5分）
- 長い休憩（デフォルト: 15分）

※設定変更はタイマー停止中のみ反映されます
